# 🏦 Bank Churn Prediction ML App

A Machine Learning web application that predicts whether a bank customer is likely to leave the bank.

## 🚀 Live Demo
https://bank-churn-predictor-krishna.streamlit.app

## 📌 Features
- Customer churn prediction
- Interactive Streamlit dashboard
- Probability prediction
- Charts and visualizations
- Clean UI

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-Learn
- Streamlit
- Matplotlib

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
